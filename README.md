# Tip How To Publish

## Build

Get SHA256 Checksum for New Script Location

```bash
curl -L https://raw.githubusercontent.com/andrewalevin/homebrew-qwhawk/main/scripts/qwhawk-git.sh | shasum -a 256
```

## Install 

```bash
brew tap andrewalevin/qwhawk

brew install qwhawk
```
